# План разработки кастомного виджета заметок с вложенностью и dnd-kit

## Общая цель
Создать легковесный редактор заметок с поддержкой:
- Вложенности (отступы, parent/child) для строк внутри заметки
- Drag-and-drop для перемещения и изменения вложенности (dnd-kit)
- Изменения типа строки (заголовок, список, ссылка, файл, текст)
- Интуитивного создания/удаления строк (Enter, Backspace)
- Сериализации структуры для хранения в БД и восстановления

---

## ВАЖНО (архитектурные требования)
- В заметке есть строки (NoteLine), которые могут вкладываться друг в друга (parentId, вложенность любого уровня до лимита).
- Каждая строка имеет тип (заголовок, список, текст, ссылка, файл и т.д.).
- Строки должны располагаться в правильном порядке относительно своего родителя (order уникален в рамках одного parentId).
- При изменении порядка/вложенности строки должны корректно обновлять parentId и order.
- Бэкенд и фронтенд должны поддерживать хранение и обработку вложенных строк с порядком.
